# Documento MDX de Teste

Este é um arquivo **MDX** (Markdown + JSX) para testar a funcionalidade de upload no MCP Vale Server.

## Conteúdo de Exemplo

O formato MDX permite combinar Markdown com componentes JSX, mas para o Vale, tratamos como texto normal.

### Problemas para Detecção

Aqui temos alguns problemas propositais:

- Repetições desnecessárias: muito muito importante
- Uso excessivo de advérbios: realmente muito interessante  
- Frases longas demais que podem ser difíceis de ler e que o Vale pode sugerir que sejam quebradas em partes menores para melhorar a compreensão do leitor
- Voz passiva: Este arquivo foi criado especificamente para testes

Teste completo do sistema de upload de arquivos!
